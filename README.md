# Projeto Integrador 2019.2 - UNICARIOCA - Rio Comprido

## Instalar dependencias do projeto
```
yarn install
```

### Hot-reloads para desenvolvimento
```
yarn run serve
```

### Compilar e minificar para produção
```
yarn run build
```

### Rodar Testes
```
yarn run test
```

### Procurar por erros nos arquivos
```
yarn run lint
```
